## Teknologi yang Digunakan

- **HTML5**
- **CSS3**
- **Bulma CSS** - framework CSS modern
- **Font Awesome** - ikon
- **Google Fonts** - Inconsolata & Inter
- **JavaScript** - toggle bahasa dan dark mode

---

## Catatan

- Semua teks yang bisa diterjemahkan menggunakan atribut `data-id` (Indonesia) dan `data-en` (English).
- Tombol dark mode dan language toggle menggunakan JavaScript untuk interaktivitas.
- Hover background di setiap item menggunakan class `hover-bg` dari CSS kustom.
- Proyek ini cocok untuk **CV online pribadi**, portofolio, atau template resume interaktif.
